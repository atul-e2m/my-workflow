Before writing any code:

1. Read and understand the full scope of the task
2. Trace how it connects to existing code — what files, functions, or modules are affected
3. If anything is unclear or ambiguous, ask me before proceeding
4. Once everything is clear, propose the implementation approach in plain English
5. Wait for my confirmation before writing a single line of code
